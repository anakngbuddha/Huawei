================================================================================
USAGE INSTRUCTIONS:
================================================================================

Adding a Task:
1. Enter task description in the input field
2. Select task type from dropdown (optional)
3. Select priority from dropdown (optional)
4. Select due date using date picker (optional)
5. Click "Add" button or press Enter

Filtering Tasks:
1. Click any filter button to apply that filter
2. Click "Reset" to return to showing all tasks
3. The active filter is highlighted in black

Task Information Display:
- Task type shows as a badge with tag icon
- Priority shows as a colored badge
- Due date shows with clock icon
- Completed tasks appear with strikethrough text
